Use Case Name Administrator Maintains Departments
Description Create: An administrator enters the name of a new location. The system checks to make sure the location name is unique and saves the new location.

Update: An administrator enters a different name for a location. The system checks to make sure the location name is unique and updates the location.

Delete: An administrator selects a location to delete. The system checks to make sure that there are no open tickets for that location and deletes the location.

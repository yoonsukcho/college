Use Case Name Administrator Maintains Service Catalogue
Primary Actor Administrator
Level Sub-function
Description Create: An administrator enters service category and service category name.

Update: An administrator enters a different service category or service category name. The system checks to make sure the service category name is unique and updates the service cateogory.

Delete: An administrator selects a service category to delete. The system checks to make sure that there are no open service item and deletes that item.
